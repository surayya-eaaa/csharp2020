Øvelse 1

Hvis du ikke har Windows på din laptop, så er det tid at få installeret dette operativ system. Licens hertil finder du på linket https://aka.ms/devtoolsforteaching hvor du kan logge ind med din skole konto klikke dig frem til gratis Windows licens.

Du kan vælge mellem:

  • Windows som eneste operativsystem på din computer.
  • Windows blandt de mulige opstartssystemer på din computer via dual boot.
  • Virtuelt miljø med Windows installation.
    På Linux har du mulighed for at installere Windows i f.eks. VirtualBox eller KVM virtuelt miljø.
    På Mac-computer kan du f.eks. vælge VirtualBox, Bootcamp eller Parallels som virtuelt miljø for installation af Windows.

Øvelse 4

Lav en konsol applikation i Visual studio

(Brug ”Console App (.Net Framework)” template)

Få den til at udskrive ”Hello World” til konsollen. 

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace ConsoleApp1
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello world!");
            Console.ReadKey();
        }
    }
}

Udvid nu applikationen med lidt mere avanceret string output. Det hele er forklaret trinvist, når du følger denne tutorial https://docs.microsoft.com/en-us/dotnet/csharp/tutorials/intro-to-csharp/hello-world 

Kør både fra Visual studio og et konsol vindue. Du finder det eksekverbare program i bin\Debug eller bin\Release folderen under dit projekt.

Øvelse 7

Opret og kør et WPF-program via Visual Studio create new project.

Eksperimenter med at ”tegne” brugergrænseflader ved hjælp af skabelonen ”Windows Presentation Foundation” – undlad at skrive koden selv. Giv solution, projekt, dialogfiler og program fornuftige navne.

Som minimum skal I prøve at ændre navnet og størrelsen på et vindue, indsætte en eller flere knapper, checkboxe, en listbox, labels og mindst en textbox. Eksperimenter med højre-/venstrestillet tekst.

Hvis I vil lave flere dialoger, så gør det i separate applikationer – vi kommer til navigering senere.

Når I har ”tegnet”, så forsøg at afvikle applikationen / applikationerne.
